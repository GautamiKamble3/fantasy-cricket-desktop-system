# Fantasy Cricket Desktop System

A desktop-based fantasy cricket system developed using Python and Qt for team creation and score calculation.

## Features
- Create fantasy teams
- Player selection by category
- Credit limit validation
- Save and view teams
- SQLite database integration
- User-friendly GUI

## Technologies Used
- Python
- PyQt / Qt Designer
- SQLite

## How to Run
1. Install Python 3.x  
2. Install dependency:
   pip install PyQt5  
3. Execute:
   python main.py

## Project Structure
- main.py : main program file
- create_db.py : database creation script
- fantasy_cricket.ui : Qt design file
- fantasy_cricket.db : project database
- Screenshots/ : UI and workflow images
